# UI Human Posture 3D Models

Original low-poly mannequin models generated for radar/posture UI usage.

Files:
- human_standing.glb / .obj
- human_sitting.glb / .obj
- human_lying.glb / .obj
- human_posture_set_all_poses.glb: all three poses together for quick preview/testing
- preview_*.png: basic previews

Coordinate convention:
- Units: meters
- Z is up
- +Y is forward
- Bottom of each model is normalized to Z = 0

Recommended UI mapping:
- Standing posture label -> human_standing.glb
- Sitting posture label -> human_sitting.glb
- Lying/fall posture label -> human_lying.glb

These are simple original geometry assets made from primitives, so you can recolor, scale, simplify, or edit them freely for your project UI.
